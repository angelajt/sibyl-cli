# sibyl-cli
command line interface for the [sibyl](https://github.com/synacor/sibyl) planning poker sprint/scrum tool

## installation
`go get -u -v github.com/angelajt/sibyl-cli`

## usage
```bash
$ sibyl-cli -h
  -d string
    deck name
  -r string
    room name
  -t string
    new room topic sentence
```
